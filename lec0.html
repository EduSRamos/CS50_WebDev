A table. Tag: "table"

  • New row: tag "tr" ("table rows")

  • Column item of a regular row: tag "td" ("table data")

A form. Tag: "form"

  • To input data, use tag "input" without end-tag
    • Attribute "type", e.g., ' "text" '
    • Attribute "placeholder"
    • Attribute "name" (gives a name to the input field). Later we'll be able to reference this field by its name.
  • A button uses the "button" tag
