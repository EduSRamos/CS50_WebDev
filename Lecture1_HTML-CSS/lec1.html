Example #3 - inheritance

All these alerts inherit from "message".

Success! < div class="success">

.success { @extend %message; background-color: green; }

Warning! < div class="warning">

.warning { @extend %message; background-color: yellow; }

Error! < div class="error">

.error { @extend %message; background-color: red; }
